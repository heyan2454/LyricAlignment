# Manual

本目录只保存已经形成稳定约定的操作与协议，不记录当前进度或临时决定。

- `data_preparation.md`：外部数据布局、manifest 与存储边界。
- `data_curation.md`：自动诊断、质量分层和人工复核协议。
- `dataset_split_and_leakage.md`：song-level split、官方 test 封存和跨数据集泄漏检查。
- `experiment_protocol.md`：raw baseline、分阶段训练和公平比较要求。
- `metric_definitions.md`：中文字符级指标口径。
- `model_usage.md`：模型来源、revision、输入输出和训练接口核验要求。
- `git_workflow.md`：Git remote、首次初始化和大文件边界。

计划新增：

- `annotation_schema.md`
- `text_normalization.md`
